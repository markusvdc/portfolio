Este é meu site de portfólio pessoal, criado para apresentar minha jornada como desenvolvedor front-end e destacar os projetos que construí ao longo do caminho. A aplicação foi desenvolvida com React, TypeScript e Sass, com foco em design responsivo, código limpo e experiência do usuário. Este projeto representa meu processo contínuo de aprendizado e minha paixão por criar experiências web modernas, funcionais e visualmente agradáveis.
